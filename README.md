# LOVE20 行动扩展协议之 LP 扩展协议

## 0. 关于协议

此扩展协议，基于 LOVE20 扩展协议标准，扩展行动参与机制，允许以 LP 参与行动，获得铸币激励。

## 1. 核心优势

### 1.1 公平分配

自动计算激励分配，无需人工参与验证

### 1.2 防止闪电贷攻击

无法在一个区块内完成质押、获取激励、并取回质押资产，可从根本上杜绝闪电贷攻击

### 1.3 长期主义激励机制

与治理票绑定，将参与资格与激励大小，与治理票占比绑定，只有长期主义者才能获得更多激励

## 2. 初始参数

### 2.1 最小治理票数

加入行动时，行动者必须持有大于等于最小治理票数的治理票数，以确保行动者参与治理。

### 2.2 退出等待区块数

距离最后一次参与超过退出等待区块数后，随时可取回参与 LP 代币

### 2.3 LP 代币地址

LP 代币合约地址

### 2.4 治理票占比倍数

在计算分数时，治理票占比会乘以治理票占比倍数，用于调整治理票在分数计算中的权重。

## 3. 行动者

### 3.1 加入行动

行动者使用 LP 代币加入行动，被验证后，获得铸币激励

参与条件：

- 首次参与：必须持有至少最小治理票数的治理票数
- 参与数量必须大于 0

### 3.2 退出行动

距离最后一次参与超过等待区块数后，随时退出行动并取回参与 LP 代币

### 3.3 铸造激励

行动者可以在验证阶段完成后，铸造行动激励，溢出激励将同步销毁。

激励根据以下公式分配：

```
理论激励 = 行动总激励 x LP代币占比
实际激励 = 行动总激励 x MIN(LP代币占比, 治理票占比 * 治理票占比倍数)
溢出激励 = 理论激励 - 实际激励
```

其中：

- LP 代币占比 = 参与 LP 代币数量 / 总参与 LP 代币数量
- 治理票占比 = 有效治理票数 / 总治理票数
